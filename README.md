Welcome to Motorcycle Buddies! 😊 This project showcases motorcycles owned by me and my friends over the years.

Visit the site on https://motorcycle-buddies.live/

## Features  
✅ Browse a collection of motorcycles  
✅ View details and stories behind each bike  
✅ User authentication (sign up, log in, profile updates)  
✅ Responsive design for mobile and desktop  
✅ Secure backend with Node.js, Express, and MongoDB

## Tech Stack  

### Frontend  
- React.js  

### Backend  
- Node.js + Express  
- MongoDB (Mongoose for database modeling)  
- JWT authentication  
- Nginx (for reverse proxy and deployment)
